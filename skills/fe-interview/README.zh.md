# fe-interview

一个 Claude Code skill，以资深前端团队 Leader 面试官的身份，分析候选人简历并生成全维度、分层级的面试题库，包含参考答案、追问方向和评分要点，输出为精美的交互式 HTML 页面。

[English](./README.md)

## 特性

- **简历驱动出题**：70% 的题目源自候选人真实简历内容，30% 覆盖行业基础与前沿趋势
- **分层级适配**：自动判定候选人级别（初级 / 中级 / 高级 / 架构），调整各维度出题深度和占比
- **全维度覆盖**：JavaScript 核心、TypeScript、React/Vue 深度、工程化与构建、性能优化、网络与安全、CSS、跨端开发、AI 与前端、系统设计、项目深挖、行为与软技能
- **紧跟 2026 技术前沿**：覆盖 React 19 / RSC / React Compiler、TypeScript 5.x、Vite 6、INP 指标、AI 与前端结合等最新技术
- **结构化答案**：每道题包含考察意图、参考答案、追问方向（答得好/答得不好两条路径）、评分要点
- **交互式 HTML 输出**：生成精美的可折叠 HTML 页面，包含候选人画像、面试策略、时间分配和强化学习路线
- **场景化出题**：强调真实工作场景，不问"什么是闭包"，而问"你在项目中哪里用到了闭包？解决了什么问题？"

## 安装

```bash
npx skills add hacxy/skills --skill fe-interview
```

## 使用方式

提供候选人简历（粘贴文本、附加文件或指向 URL），然后让 Claude 生成面试题：

```
帮我看看这个简历该问什么
```

```
根据这份简历生成面试题
```

```
帮我准备面试，这是候选人简历
```

Skill 会：

1. 分析简历，构建候选人画像（级别、技术栈、优势、薄弱环节）
2. 生成 15-20 道结构化面试题，每道题配完整参考答案
3. 渲染为交互式 HTML 页面并在浏览器中打开

## 输出内容

生成的 HTML 页面包含：

| 模块 | 内容 |
| --- | --- |
| 候选人画像 | 级别、技术栈标签、优势分析、潜在薄弱点 |
| 面试策略 | 时间分配彩色条、重点考察方向 |
| 面试题库 | 按分类组织的题目，可折叠的参考答案、追问方向和评分要点 |
| 强化学习路线 | 分优先级的学习方向，附推荐资源 |
