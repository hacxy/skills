# resume-review

[English](./README.md)

评审技术简历并给出专业优化建议，深耕中国互联网行业尤其是前端方向。

## 功能说明

本 skill 扮演一位拥有丰富互联网行业招聘经验的技术面试官和简历顾问，熟悉字节、阿里、腾讯、美团、快手等大厂的招聘标准。它会从 8 个维度评审简历，逐一打分，识别关键问题，并输出一份优化后的简历版本。

核心能力：

- **8 维度评审**：整体结构、个人信息、专业技能、项目经验、技术深度、技术时效性、差异化亮点、措辞与表达
- **有据可依的反馈**：每条建议都说清"为什么不好"和"怎样改好"，引用行业最佳实践
- **STAR 法则改写**：将模糊的项目描述用 STAR 框架重写，补充量化指标
- **行业对标分析**：搜索最新的前端行业简历标准和优秀范例，给出有根据的对比建议
- **输出优化版简历**：生成完整的改进版本，保留候选人真实信息

**触发词：** "简历评审"、"帮我看看简历"、"优化简历"、"简历怎么改"、"帮我改简历"、"简历诊断"、"review my resume"
