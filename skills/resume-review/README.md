# resume-review

[中文](./README.zh.md)

A skill for reviewing technical resumes and providing professional optimization suggestions, with deep expertise in China's internet industry, especially frontend development positions.

## What it does

This skill acts as an experienced technical interviewer and resume consultant who is familiar with the hiring standards of major Chinese internet companies (ByteDance, Alibaba, Tencent, Meituan, Kuaishou, etc.). It reviews resumes across 8 dimensions, scores each one, identifies critical issues, and produces an optimized version of the resume.

Key capabilities:

- **8-dimension review**: Structure, personal info, skills, project experience, technical depth, tech currency, differentiation, and writing quality
- **Evidence-based feedback**: Every suggestion includes "why it's a problem" and "how to fix it", with references to industry best practices
- **STAR method rewriting**: Rewrites vague project descriptions using the STAR framework with quantified metrics
- **Industry benchmarking**: Searches for the latest resume standards and examples in China's frontend industry to provide grounded comparisons
- **Optimized resume output**: Generates a complete improved version while preserving authentic information

**Triggers:** "review my resume", "resume review", "optimize resume", "check my CV", "简历评审", "帮我看看简历", "简历优化", "简历怎么改", "帮我改简历"
