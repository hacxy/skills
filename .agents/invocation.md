# Invocation Guide

How to invoke skills in this repository.

## Skill Invocation

Skills can be invoked in two ways:

### User-Invoked

Type the skill name directly:
```
/skill-name
```

### Auto-Invoked

Some skills are triggered automatically by the agent when the task matches the skill's description.

## Skill Discovery

Skills are discovered from:
- `.agents/skills/` (project-level)
- `~/.agents/skills/` (global-level)
