# Skills

[中文](./README.zh.md)

A collection of agent skills compatible with all platforms supported by [skills.sh](https://skills.sh/), including Claude Code, Cursor, Codex, GitHub Copilot, Windsurf, Gemini, Cline, and more.

## Skills

### code-review

Evaluate code quality in a sub-agent for specified files or the entire project. Outputs a structured report with findings and optimization suggestions.

**Triggers:** "review code", "code review", "analyze code", "check code quality", "evaluate code"

```bash
npx skills add hacxy/skills --skill code-review
```

## Installation

All skills in this collection can be installed via [skills.sh](https://skills.sh/):

```bash
npx skills add hacxy/skills --skill <skill-name>
```
